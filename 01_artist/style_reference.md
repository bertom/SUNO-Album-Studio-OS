## How to use this file

- Define **groove families** — feel and production, not genre labels.
- Each family gets: instruments, tempo feel, vocal treatment, exclude list, example album roles.
- Agents read `style_reference_summary.md` first; this file is the deep reference.
- Never put artist names in SUNO fields — translate families to concrete musical language.
