# Contributing

Thanks for improving SUNO Album Studio OS.

## What belongs here

- Workflow improvements, clearer docs, better templates
- SUNO hygiene learnings that help **any** musician
- CLI fixes and tests in `06_integrations/suno/`

## What does not belong

- Your personal albums, masters, or artist identity (keep those in your fork's `01_artist/` and `02_albums/`)
- Artist-specific catalog entries in upstream PRs
- Secrets or API keys

## How to contribute

1. Fork the repo
2. Make focused changes — one concern per PR when possible
3. Open an issue first for large workflow changes
4. Run `./studio-suno validate` on `_album_template` if you touch CLI or SUNO docs

## Tone

Warm, practical, musician-first — not corporate compliance docs.
